O que é HTML?

O HTML (HyperText Markup Language) é a linguagem base para se desenvolver qualquer site. É uma linguagem baseada em marcação, onde marcamos os elementos para definir quais informações a página vai exibir.

O que é CSS?

O CSS é uma linguagem de folha de estilo composta por “camadas”, criado com o propósito de estilizar as páginas HTML, ou seja, definir a aparência das páginas, para deixá-las visualmente mais bonitas e agradáveis, podendo alterar a fonte, cor, layout e muito mais.

História

Abaixo segue um breve resumo da criação e desenvolvimento do HTML e CSS

HTML

O HTML foi criado em 1991, por Tim Berners-Lee, no CERN (European Council for Nuclear Research) na suíça. Inicialmente o HTML foi projetado para interligar instituições de pesquisa próximas, e compartilhar documentos com facilidade.

CSS

O Cascading Style Sheets (CSS) foi proposto pela primeira vez em Outubro de 1994, por Hakon Lie, que queria facilitar a programação de sites, que na época era muito mais complexa. Em 1995 o CSS1 foi desenvolvido pela W3C, um grupo de empresas do ramo da informática.

Conclusão

Devido aos inúmeros benefícios, podemos dizer que a criação do HTML e CSS facilitou muito a vida dos desenvolvedores WEB.

Vantagens

Abaixo segue algumas vantagens do HTML e CSS.

HTML

Amplamente utilizado, fácil de aprender e usar e também é leve e rápido de carregar.

CSS

Reduz o código HTML, pode ser aplicado a diferentes documentos e tem maior eficiência no gerenciamento do Layout.

Conclusão

Aprender HTML e CSS foi o inicio da jornada de muitos desenvolvedores, e para quem deseja seguir carreira no desenvolvimento Web são os primeiros passos a serem dados.
